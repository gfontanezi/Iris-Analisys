# Classificação do Dataset Iris 🌸

Este projeto utiliza o conjunto de dados **Iris** para treinar e avaliar modelos de Machine Learning capazes de classificar flores em três espécies diferentes com base em características físicas.

O projeto foi desenvolvido em Python utilizando Jupyter Notebook.

---

## 📊 Dataset

O dataset Iris contém 150 amostras de flores, com as seguintes variáveis:

- SepalLength (comprimento da sépala)
- SepalWidth (largura da sépala)
- PetalLength (comprimento da pétala)
- PetalWidth (largura da pétala)
- Species (espécie da flor)

As espécies são:
- Iris-setosa  
- Iris-versicolor  
- Iris-virginica  

---

## 🧠 Objetivo

Treinar um modelo de Machine Learning para:
- Prever a espécie da flor a partir das medidas fornecidas
- Avaliar o desempenho do modelo
- Visualizar os resultados

---
